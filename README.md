# modulo-autentication-snow-boys
proyecto enfocado para planificar el desarrollo de un modulo de autenticacion usando github projets.

##objetivo del modulo
tiene como objetivo desarrollar un modulo de autenticacion el cual permitira a los usuarios registrarse, iniciar sesion, y ingresar al dashboard a demas que se uso github projets para planificar, organizar y tener un flujo de trabajo mas agil

## Historias de usuarios

### Historia de usuario 1 - Inicio de sesion 
Como usuario registrado quiero iniciar sesión con mi correo y contraseña para acceder al panel de control de la aplicación.

### Historia de usuario 2 - registro de usuario
Como visitante no registrado quiero crear una cuenta proporcionando mi nombre, correo electrónico y una contraseña segura para poder acceder a sus funciones exclusivas.

### Historia de usuario 3 - Dashboard
como usuario autenticado quiero ser dirigido automáticamente a un dashboard para visualizar un resumen de mis actividades, navegar por la plataforma y tener la opción de cerrar sesión
